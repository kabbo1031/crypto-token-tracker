# Simple Crypto Token Tracker

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- Read a token balance
- Show token name and symbol
- Learn how smart contracts work

This project is good for:
- Web3 beginners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- A web browser
- MetaMask wallet
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Connect your wallet
5. Enter a token contract address
6. Click "Check Token"

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Token settings  
- `token.js` = Token reader  
- `wallet.js` = Wallet connect  
- `ui.js` = Screen updates  
- `abi.js` = Token contract ABI  
- `utils.js` = Helpers  
- `style.css` = Simple style  
- `logger.js` = Logs  

---

- Learn how to read smart contracts

- Learn how wallets interact with apps
